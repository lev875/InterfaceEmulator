Функция расчета CRC16.

Расчитывает контрольную сумму пакета.

Warning
Использовать только эту функцию, иначе не сойдется!
  • Name : CRC-16 CCITT
  • Poly : 0x1021 x^16 + x^12 + x^5 + 1
  • Init : 0xFFFF
  • Revert: false
  • XorOut: 0x0000
  • Check : 0x29B1 ("123456789")
  • MaxLen: 4095 байт (32767 бит)
    Обнаружение одинарных, двойных, тройных и всех нечетных ошибок
